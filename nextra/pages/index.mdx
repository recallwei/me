## Get Started 🌈

import { StarCount } from "../components"

export const getStaticProps = async () => {
  const res = await fetch("https://api.github.com/repos/recallwei/recallwei")
  const data = await res.json()
  return {
    props: {
      ssg: {
        starCount: data.stargazers_count
      }
    },
    revalidate: 60
  }
}

This is my personal wiki website, has <StarCount /> stars 🌟 on GitHub! I want to record anything, not limited to code, books, music, movies, food, travel, etc.
