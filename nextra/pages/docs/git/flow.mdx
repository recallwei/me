# 工作流

## 克隆仓库

### 从 GitHub 克隆仓库

```bash filename="Bash"
git clone http://github.com/recallwei/recallwei
```

### 克隆仓库并在本地重命名

```bash filename="Bash"
git clone http://github.com/recallwei/recallwei my-project-name
```

## 将文件添加至暂存区

### 添加所有文件

```bash filename="Bash"
git add .
```
