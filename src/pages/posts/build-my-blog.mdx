## 建立 Astro 项目

```bash
pnpm create astro@latest
```

## 前端工程化配置

### 添加 CSpell

```bash
cspell "**"
```

### 添加 Prettier

使用 [prettier-plugin-astro](https://github.com/withastro/prettier-plugin-astro/)

注意 pnpm 执行 Prettier 格式化命令需要额外配置。
