## 克隆仓库

### 从 GitHub 克隆仓库

```bash
git clone http://github.com/recallwei/recallwei
```

### 克隆仓库并在本地重命名

```bash
git clone http://github.com/recallwei/recallwei my-project-name
```

## 将文件添加至暂存区

### 添加所有文件

```bash
git add .
```

## 提交文件

```bash
git commit -m "Commit Message"
```

## 推送至远程仓库

```bash
git push
```

## 从远程仓库拉取

```bash
git pull
```
