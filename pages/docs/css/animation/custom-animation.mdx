# 自定义动画

## Tada

```css filename=".css" showLineNumbers
.tada {
  animation: tada infinite 2s;
}

@keyframes tada {
  0%,
  100% {
    transform: rotate(0);
  }
  10%,
  30%,
  50%,
  70%,
  90% {
    transform: rotate(-3deg);
  }
  20%,
  40%,
  60%,
  80% {
    transform: rotate(3deg);
  }
  100% {
    transform: rotate(0);
  }
}
```
